Docker start

Docker start and docker stop

1.List running container

docker ps -a -f status=running

alternatively

docker container ls -a -f status=running

2.Stop and remove the running container

docker container stop {containerID}

docker rm {containerID}

alternatively

docker rm -f {containerID}

3.Docker start and docker stop

docker image ls

Start the container

docker run -d -p 80:80 {containerID}

Stop the container

docker rmi {containerID}
